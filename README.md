## 基于 SpringBoot + Thyemeleaf 的个人博客

### 1.技术栈

#### 后端

- 核心框架: Springboot 2.1.3
- 数据库: MySQL 8.0.13
- 连接池: Druid
- 缓存: Redis + SpringBoot-Cache
- MarkDown处理: Flexmark
- 其他: Lombok

#### 前端

- Thymeleaf
- Bootstrap 3
- JQuery
- MarkDown编辑: Editor.md
- 弹窗: SweetAlert
